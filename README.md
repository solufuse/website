# Solufuse Website

Ce repository contient le site web principal (Landing Page / Dashboard) de Solufuse.

## Hébergement
Ce site est statique (HTML/JS). Il peut être hébergé sur :
- **GitHub Pages** (Recommandé : Allez dans Settings > Pages > Source: main branch)
- **Dokploy**
- Ou n'importe quel serveur web statique.

## Configuration
Le fichier `index.html` contient une section `const firebaseConfig` qu'il faut remplir avec les clés Firebase du projet pour que l'authentification fonctionne.
